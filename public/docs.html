📖 Pengenalan

Kos Manager adalah aplikasi manajemen kos / boarding house lengkap, dibangun di atas Laravel 13 + Filament v5. Aplikasi ini mengelola seluruh aspek operasional bisnis kos dari satu dashboard:

🏢

Multi Properti & Kamar

Kelola banyak kos dengan ratusan kamar dari satu admin panel.

📄

Kontrak & Tagihan

Generate invoice bulanan otomatis, denda keterlambatan, reminder WhatsApp.

💳

Payment Gateway

Integrasi Midtrans, Tripay, atau pembayaran manual transfer bank.

📱

Portal Penyewa

Penyewa lihat tagihan, bayar online, lapor maintenance lewat HP.

🌐

Landing Page

Halaman publik untuk calon penyewa cari kamar & booking online.

📊

Laporan Keuangan

Grafik pendapatan, occupancy rate per properti, tunggakan.

📦 Instalasi

1

Clone / Extract Source Code

Letakkan source di folder web server Anda, mis. /var/www/kos atau htdocs/kos_app.

cd /var/www/
unzip kos-manager.zip
cd kos-manager
2

Install Dependencies

composer install --no-dev --optimize-autoloader
npm install
npm run build
3

Salin & Edit File .env

cp .env.example .env
php artisan key:generate

Edit minimal field-field berikut di .env:

APP_NAME="Kos Manager"
APP_URL=https://kos.domain.anda

DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=kos_manager
DB_USERNAME=root
DB_PASSWORD=rahasia

MAIL_MAILER=smtp
MAIL_HOST=smtp.gmail.com
MAIL_PORT=587
4

Migrasi Database & Seed

php artisan migrate --force
php artisan db:seed --force
php artisan storage:link

💡 Tips: Untuk demo dengan data lengkap (100 kos, 10K kamar, 1K user), jalankan: php artisan db:seed --class=MassDataSeeder

5

Setup Permissions (Linux)

chown -R www-data:www-data storage bootstrap/cache
chmod -R 775 storage bootstrap/cache
6

Setup Cron Job

Untuk reminder otomatis, generate tagihan bulanan, dll. Tambahkan ke crontab:

* * * * * cd /var/www/kos && php artisan schedule:run >> /dev/null 2>&1

🔑 Aktivasi Lisensi

Aplikasi memerlukan lisensi aktif dari whitelabel.co.id untuk dijalankan. Ada dua cara aktivasi:

Cara 1: Via Command Line (Direkomendasikan)

Setelah membeli lisensi, Anda akan menerima activation key dengan format XXXXX-XXXXX-XXXXX-XXXXX.

php artisan license:activate XXXXX-XXXXX-XXXXX-XXXXX

Output sukses:

✓ Lisensi berhasil diaktifkan untuk Kos Manager v1.0.0
  Tipe: Single Domain
  Aktif sampai: 2027-04-26

Cara 2: Via Admin Panel

  1. Buka /admin/license-settings
  2. Masukkan activation key Anda
  3. Klik tombol Aktifkan
  4. Status berubah jadi Aktif

Validasi & Manajemen Lisensi

Perintah CLIFungsi
php artisan license:infoLihat info lisensi aktif
php artisan license:validateCek ulang validitas ke server
php artisan license:revokeCabut lisensi (sebelum pindah domain)

⚠️ Penting: Sebelum pindah server / domain, jalankan license:revoke dulu agar bisa diaktifkan di domain baru. Lisensi terikat per domain.

📲 WhatsApp Gateway (Fonnte)

Untuk reminder tagihan otomatis & blast pesan ke penyewa, integrasikan dengan Fonnte:

  1. Daftar dan beli paket di fonnte.com
  2. Hubungkan WhatsApp Anda di dashboard Fonnte
  3. Copy API Token
  4. Buka /admin/general-settings section "Notifikasi WhatsApp"
  5. Aktifkan toggle, paste API Key & nomor sender

Fitur WhatsApp Otomatis

  • 📋 Kirim invoice baru ke penyewa
  • ⏰ Reminder H-3 sebelum jatuh tempo
  • 🔴 Notifikasi tagihan jatuh tempo (overdue)
  • 🏠 Welcome message + kredensial portal saat penyewa baru daftar

📋 Kontrak Sewa (Lease)

Buka /admin/leases. Menghubungkan Kamar dan Penyewa dengan periode + harga.

Alur Kontrak

1
Pilih kamar (status available) →
2
Pilih penyewa →
3
Set tanggal mulai & berakhir →
4
Status kamar otomatis jadi occupied

Billing Cycle

  • Monthly (1 bulan) — paling umum
  • Quarterly (3 bulan)
  • Yearly (12 bulan, biasanya diskon)

Status Kontrak

  • pending — draft, belum dimulai
  • active — sedang berjalan
  • expired — periode habis (otomatis via cron)
  • terminated — dihentikan paksa (penyewa keluar)

Cek Kontrak Hampir Habis

php artisan lease:check-expiring     # default H-30
php artisan lease:check-expiring 7   # H-7

📱 Portal Penyewa (End-User)

Bagian ini untuk Anda berikan ke penyewa Anda sebagai panduan menggunakan aplikasi.

🎯 Cara Penyewa Login

  1. Buka URL https://kos.domain.com/portal/login di HP / browser
  2. Masukkan Nomor HP (yang didaftarkan admin) — contoh: 081234567890
  3. Masukkan password (default: password — bisa diubah di menu Profil)
  4. Klik Masuk

Menu Portal

🏠 Dashboard

  • Info kamar yang disewa (properti, no kamar, kontrak)
  • Jumlah tagihan belum bayar
  • Jumlah laporan maintenance terbuka
  • Quick link ke pembayaran & lapor maintenance

📄 Tagihan

  • Riwayat semua tagihan
  • Status: Lunas / Belum Bayar / Jatuh Tempo
  • Detail invoice (rincian biaya, diskon)
  • Tombol bayar online (jika gateway aktif)

🔧 Maintenance

  • Riwayat laporan kerusakan
  • Tombol "Lapor Baru"
  • Status: Menunggu / Diproses / Selesai

👤 Profil

  • Edit nama, HP, email, WhatsApp
  • Ganti password portal

Cara Lapor Kerusakan

  1. Login portal → klik menu Maintenance
  2. Klik tombol + Lapor Baru
  3. Isi judul (mis. "AC tidak dingin")
  4. Deskripsikan masalah secara detail
  5. Pilih prioritas: Rendah / Sedang / Tinggi / Urgent
  6. Klik Kirim Laporan
  7. Admin akan merespon, status berubah otomatis

Cara Ganti Password

  1. Login portal → menu Profil
  2. Scroll ke section "Ubah Password"
  3. Isi password lama, baru (min 6 karakter), dan konfirmasi
  4. Klik Ubah Password

🌐 Landing Page Publik

URL: https://kos.domain.com/ — bisa diakses tanpa login. Tampilan untuk calon penyewa.

Yang Tampil Otomatis

  • Daftar semua properti dengan status is_active = true
  • Foto, nama, alamat, kota
  • Jumlah kamar tersedia (dihitung dari rooms status=available)
  • Range harga termurah – termahal
  • Testimoni penyewa (max 6)
  • Kontak global (WhatsApp, telepon, email, alamat)

Single vs Multi Property

Jika hanya ada 1 properti aktif → langsung redirect ke /property/{id} (page detail).

Jika ada lebih dari 1 properti → tampil sebagai grid pilihan.

🎫 Booking Online

Calon penyewa bisa booking via tombol di halaman properti. Form: /booking/{property_id}.

Alur Booking

1
Calon penyewa isi form: nama, HP, email, kapan move-in, pilih tipe kamar, pesan
2
Admin terima notifikasi, buka /admin/booking-requests
3
Klik tombol Hubungi via WhatsApp di list — buka chat WA langsung dengan calon penyewa
4
Setelah deal, buat Penyewa baru → Kontrak → status booking jadi converted

Status Booking

  • pending → baru masuk
  • contacted → sudah dihubungi admin
  • approved → setuju, menunggu deposit
  • converted → jadi kontrak resmi
  • rejected → ditolak

🖨 Cetak Tagihan & Kwitansi

Aplikasi ini punya 3 dokumen yang bisa dicetak / di-download dalam format PDF (pakai engine DomPDF):

📄

Tagihan / Invoice

Untuk setiap invoice — semua status (draft, sent, paid, overdue).

Berisi: header app, info penyewa, periode, breakdown biaya, total, status badge.

🧾

Kwitansi

Bukti pembayaran — hanya untuk invoice yang sudah paid.

Berisi: stempel "LUNAS", terbilang, signature line untuk pembayar & pengelola.

📊

Laporan Tagihan

Rekap list tagihan dengan filter periode & status.

Berisi: tabel semua invoice, total grand sum di footer.

Cara cetak dari Admin Panel

  1. Buka /admin/invoices
  2. Di tiap baris, ada tombol "Cetak Tagihan" (ikon printer abu-abu) — buka tab baru, langsung tampil PDF preview di browser. Klik ikon download di PDF viewer untuk simpan.
  3. Tombol "Cetak Kwitansi" (ikon centang hijau) hanya muncul kalau invoice sudah paid.
  4. Untuk laporan periode, klik tombol "Cetak Laporan PDF" di pojok kanan atas list (di samping tombol "Buat Tagihan").

Cara cetak dari Portal Penyewa

  1. Penyewa login di /portal/login
  2. Buka menu Tagihan → klik salah satu invoice
  3. Di halaman detail, ada 2 tombol:
    • Cetak Tagihan (biru) — selalu tersedia
    • Cetak Kwitansi (hijau) — muncul kalau status sudah Lunas

URL Langsung (untuk integrasi/embed)

DokumenURLAkses
Invoice PDF /print/invoice/{id} Admin (semua) atau penyewa (milik sendiri)
Kwitansi PDF /print/kwitansi/{id} Sama, hanya invoice paid
Laporan Tagihan PDF /print/report/invoices Admin login

Tambahkan query ?download=1 untuk paksa download (bukan preview), ?status=paid / ?from=2026-01-01 / ?to=2026-12-31 untuk filter laporan.

# Contoh
/print/invoice/123                              → preview PDF di browser
/print/invoice/123?download=1                   → download langsung
/print/kwitansi/123                             → kwitansi (hanya untuk paid)
/print/report/invoices?status=overdue           → laporan tunggakan
/print/report/invoices?from=2026-01-01&to=2026-12-31&status=paid&download=1

💡 Tips cetak fisik: Saat preview PDF di browser, klik ikon printer di toolbar PDF viewer (atau tekan Ctrl+P) untuk kirim ke printer. Format A4 portrait sudah otomatis.

Customisasi tampilan PDF

Edit file blade berikut sesuai kebutuhan branding Anda:

  • resources/views/pdf/invoice.blade.php — desain invoice
  • resources/views/pdf/kwitansi.blade.php — desain kwitansi (warna hijau, stempel LUNAS)
  • resources/views/pdf/report-invoices.blade.php — desain laporan

Logo & nama aplikasi otomatis ambil dari General Settings (setting('app_name'), setting('contact_address')).

🆘 Troubleshooting

"Lisensi Tidak Valid" muncul terus

Pastikan:

  • Server bisa konek internet ke whitelabel.co.id
  • Domain di .env APP_URL sama dengan domain yang Anda akses
  • Cache lisensi di-clear: php artisan cache:forget license_valid
  • Jalankan ulang: php artisan license:validate
Foto upload tidak tampil di landing

Buat symlink storage:

php artisan storage:link

Pastikan folder public/storage mengarah ke storage/app/public.

Webhook Midtrans / Tripay tidak masuk
  • URL webhook harus public (bukan localhost) — pakai ngrok untuk testing
  • Pastikan Server Key di settings admin sama dengan dashboard provider
  • Cek log: storage/logs/laravel.log
Penyewa lupa password portal

Reset dari CLI:

php artisan portal:set-password 081234567890 password_baru

Atau dari admin: /admin/occupants/{id}/edit → field Password Portal.

Error 500 di admin / portal setelah update

Clear cache & cek log:

php artisan optimize:clear
tail -50 storage/logs/laravel.log
php artisan app:smoke-test --verbose-errors
Tagihan tidak ter-generate otomatis

Pastikan cron jalan:

* * * * * cd /var/www/kos && php artisan schedule:run >> /dev/null 2>&1

Cek: php artisan schedule:list

Manual generate: php artisan invoice:generate-monthly

Plugin error → seluruh aplikasi tidak bisa diakses

Plugin yang rusak (config hilang, syntax error) akan crash seluruh app. Solusi:

  1. Cek log: tail -50 storage/logs/laravel.log
  2. Nonaktifkan plugin via DB langsung: 
    UPDATE installed_plugins SET is_active = 0 WHERE plugin_slug = 'finance';
  3. Atau perbaiki file plugin lalu php artisan optimize:clear
Cara menghilangkan popup "Pembelian Aplikasi" di landing page

Popup di pojok kanan bawah landing page (/ dan halaman properti) bisa dihilangkan dengan beberapa cara:

Cara 1: Tutup sementara (klik tombol X)

Klik tombol × di pojok kanan atas popup. Pilihan disimpan di localStorage browser dan popup tidak muncul lagi selama 7 hari di browser yang sama.

Cara 2: Hilangkan untuk satu browser saja (DevTools)

Buka DevTools (F12) → tab Console → jalankan:

localStorage.setItem('purchasePopupDismissed', '9999999999999')

Cara 3: Hilangkan permanen untuk semua pengunjung

Edit file resources/views/landing/layouts/app.blade.php, hapus seluruh blok yang ditandai dengan komentar Blade {{-- PURCHASE POPUP --}} sampai {{-- /PURCHASE POPUP --}} (di dekat akhir file, sebelum </body>). Setelah itu jalankan:

php artisan view:clear

Cara 4: Sembunyikan via CSS (cepat, tanpa edit blok)

Tambahkan rule ini di file resources/views/landing/layouts/app.blade.php di dalam <style> manapun:

#purchase-popup { display: none !important; }
